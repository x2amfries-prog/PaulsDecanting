# Paul's Decanting - ScentSplitter

A luxury fragrance decants e-commerce website built with HTML, CSS, and Tailwind CSS.

## Features
- Modern dark theme design
- Luxury gold accents
- Responsive navigation
- Shopping cart functionality

## Live Site
Visit: https://x2amfries-prog.github.io/PaulsDecanting

## How to Deploy to GitHub Pages
1. Go to your repository settings
2. Navigate to "Pages" section
3. Select "main" branch as source
4. Your site will be live in a few moments!
